## Overview

This repository contains the data, code, and analysis for our final report on environmental decoupling in Norwegian salmon aquaculture. We apply two IPAT decompositions to assess whether absolute decoupling has occurred between salmon production growth and two biodiversity pressures:

1. **IPAT 1 — Wild fish extraction** (*W*, tonnes/year): total wild forage fish extracted to supply Norwegian salmon feed (2000–2013).  
2. **IPAT 2 — Persistent antiparasitic chemical discharge** (*A_out*, kg/year): total persistent antiparasitic substances discharged into Norwegian fjords from sea lice treatments (2012–2019).

## Key Findings

| Pressure | Period | Production change | Pressure change | Dominant driver |
|----------|--------|-------------------|-----------------|-----------------|
| Wild fish extraction (*W*) | 2000–2013 | +166% | **−64%** | W/M (FFDR_FM): −0.94 ln-points |
| Chemical discharge (*A_out*) | 2012–2019 | +10.7% | **−95.3%** | δ (bath→oral shift): −2.04 ln-points |

## Reproducing the Analysis

### Requirements

- **R** (≥ 4.1) with packages: `tidyverse`, `readxl`, `scales`, `patchwork`, `knitr`
- Intermediate years in IPAT 1 are obtained by piecewise linear interpolation (`approx()` in R)

### Running the code

1. Open `code/ipat1_wild_fish_extraction.Rmd` in RStudio. Set the working directory to `data/`. Knit to PDF.
2. Open `code/ipat2_antiparasitic_discharge.Rmd` in RStudio. Set the working directory to `data/`. Knit to PDF.

Both scripts produce all figures used in the report and verify the IPAT identities (residual = 0% for all years).

## IPAT Identities

**IPAT 1:**
$$W = P \times \frac{F}{P} \times \frac{M}{F} \times \frac{W}{M}$$

**IPAT 2:**
$$A_{out} = N_{salmon} \times \frac{N_{lice}}{N_{salmon}} \times \frac{Tr_{total}}{N_{lice}} \times \frac{Tr_{chem,pol}}{Tr_{total}} \times \frac{A_{out}}{Tr_{chem,pol}}$$

where *A_out = A_bath,excl.H₂O₂ × 1.0 + A_oral × 0.03*. H₂O₂ is excluded due to non-persistence (half-life < 1 h in seawater).
